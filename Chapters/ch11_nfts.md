# 第十一章：非同质化代币 (NFTs)

想象一下，为一个任何人都可以“右键另存为”的 JPEG 图片支付 7000 万美元。这听起来荒谬至极。整个前提似乎违背了我们对价值的所有认知：如果某样东西可以零成本完美复制，它怎么可能价值数百万美元？然而，2021 年 3 月，这一幕在享誉世界的艺术品拍卖行佳士得（Christie's）真实上演了——Beeple 的作品《Everydays》以这一价格卖给了名为 Metakovan 的买家。他购买的并不是对像素的独占访问权，相反，他买到了一些更有趣的东西：一份经过加密验证的、证明他拥有该“原作”的证明。

## 第一节：数字所有权革命

那次 7000 万美元的购买之所以成为可能，是因为数字资产的运作方式发生了根本性的转变。与每个单位都完全相同的同质化代币不同，NFT 是**非同质化（Non-fungible）**的：每个代币都是独特的，这创造了价格发现按资产逐一进行市场。反过来，这也使得附加在每个代币上的信息（称为其**元数据 Metadata**）变得尤为重要。

### NFT 究竟是什么

在 NFT 出现之前，数字世界有一个根本缺陷：完美的复制性。任何人都可以下载高分辨率图像，创建一个与原作毫无二致、像素级精确的副本。如果副本是免费且完全相同的，那么谁能真正“拥有”一张数字图像呢？

NFT 通过将数字财产拆解为独立的、可验证的层级来解决这个问题。图像文件本身仍然可以自由复制，但 NFT 的购买授予了几个不同的组成部分：

*   **代币控制权**：区块链不可篡改地记录了某位持有者控制着 \#1234 号 NFT。
*   **来源（Provenance）**：一份证明该代币来自创作者钱包的证书，确立了真实性。
*   **使用权**：一个独立的许可协议（通常在链下），定义了持有者可以用该内容做什么。
*   **效用访问权**：智能合约可以根据代币的持有情况授予权限（如代币准入门槛功能等）。

这些层级可以独立编程，提供了极大的灵活性。与仅仅挂在墙上的画作不同，NFT 可以随时间推移而演变，在支持的情况下为创作者提供**版税（Royalties）**，与其他数字资产互动，甚至控制自己的钱包。拥有者可能持有一个 NFT，该 NFT 授予其在商业活动中使用艺术品的商业权利，而图像本身存储在分布式存储网络中，其来源则锚定在创作者的钱包中。这些层级保持模块化，并可与其他系统组合。

### 独特性是如何实现的

其核心解决方案非常简单。像 ERC-20（第二章介绍的同质化代币标准）这样的常规代币就像完全相同的美元钞票，但 NFT 就像带有编号的宝可梦卡片。每个 NFT 在其智能合约中都会获得一个唯一的标识符，区块链维护着一个永久的账本，映射哪个钱包控制哪个代币。相同的标识符编号可以存在于不同的集合中，但“集合地址 + 标识符”的组合在全球范围内是唯一的，从而确保每个 NFT 在整个区块链上都是唯一可识别的。

一项较新的创新允许 NFT 自身拥有资产。每个 NFT 都可以链接到它自己的钱包，该钱包不由私钥控制，而是由谁持有该 NFT 谁就拥有控制权。这意味着 NFT 可以随着时间的推移积累资产。一个游戏角色 NFT 可能会在你玩游戏时收集装备和货币；一个会员 NFT 可能会积累活动勋章和奖励。当你出售或转让该 NFT 时，它所拥有的一切都会自动随之转让。

然而，这种技术基础引入了一个每个项目都必须面对的核心设计张力：哪些内容留在链上，哪些内容放在链下。除了技术决策，这种灵活性还在所有权权利的实际运作方式上引入了一个根本性的法律空白。

### 版权之困

将权利拆解为独立的层级创造了显著的法律灰色地带。虽然 NFT 证明了对代币的占有，但底层艺术品的使用权受链下许可和传统版权法的管辖，而传统法律往往难以处理去中心化资产。这些许可在不同司法管辖区的可执行性尚未经过法院的严格测试，这引发了关于持有者真正能对他们价值数百万美元的 JPEG 图片做些什么的疑问。

这种模糊性导致了 NFT 领域重大的战略分歧。一些项目，如无聊猿（Bored Ape Yacht Club），授予持有者商业权利，但保留了显著的知识产权控制权。与之直接对立的是，一场强大的运动拥护通过 **Creative Commons Zero (CC0)** 将艺术品贡献给公共领域。像 Nouns DAO 和 CrypToadz 这样的项目以采用“不保留任何权利”的方式而闻名，允许任何人使用、重新创作和商业化他们的艺术作品。他们的论点是，当品牌是开放且无需许可的时，它会变得更有价值，其功能就像一个任何人都可以基于其构建的协议。在封闭的、中心化控制的品牌与开放的、去中心化的品牌之间做出选择，已成为 NFT 创作者面临的一个根本性的意识形态分歧点。

关于所有权权利的这些哲学和法律问题之所以重要，正是因为 NFT 是复杂的技术制品，其价值完全取决于创作者如何解决底层基础设施的挑战。

## 第二节：超越简单的所有权

### 存储解决方案

创建 NFT 会立即引发一个技术难题：为了最大限度地保证永久性而将所有内容存储在链上，但需支付昂贵的 Gas 费；或者为了经济实惠而将大部分内容存储在链下，但面临几年后 NFT 指向死链接的风险。

大多数项目选择混合方法。区块链记录代币所有权，并包含一个链接，指向一个包含代币名称、描述、图像和属性的文件。所有权注册表变得不可篡改，而实际内容则取决于外部存储是否持续可用。

目前已出现了一系列存储解决方案：

*   **中心化服务器**：最便宜且最灵活，但如果服务器关闭，NFT 将变得无法访问。
*   **IPFS (星际文件系统)**：内容寻址的分布式存储，文件通过其内容哈希来识别，使其更难丢失，但需要持续的“固定（Pinning）”才能保持可用（在第十三章的 DePIN 存储部分也有讨论）。
*   **Arweave**：通过捐赠基金（“永久网 Permaweb”）一次性支付费用以实现永久存储；前期成本较高。
*   **链上存储**：最高的永久性和抗审查性（例如 Autoglyphs），但单张图像的 Gas 费可能高达数千美元。

更复杂的 NFT 系列采用多层方法。它们使用内容寻址的 URI（IPFS/Arweave 哈希）来确保文件不会被悄悄更改。它们将关键的溯源信息直接存储在链上，并雇用多个固定服务提供商作为备份。有了可靠的存储基础设施，NFT 就从静态图像演变为可编程的动态资产。

### 高级代币类型

**动态 NFT (Dynamic NFTs)** 会随时间演变。一张体育明星卡 NFT 可能会在每场比赛后自动更新球员数据；数字艺术作品可能会根据持有者所在城市的天气数据改变颜色；游戏角色会积累经验值并升级，其外观和能力也会相应改变。代币本身变成了一个活生生的实体，对周围的世界做出反应。

**可组合 NFT (Composable NFTs)** 创造了财产层级：即包含其他代币的代币。想象一下购买一块虚拟土地（一个 NFT），它包含一栋房子（另一个 NFT），房子里装满了家具（更多的 NFT）。当持有者出售这块土地时，如果该系列设计支持这种嵌套，内部的所有东西都可以一起转让。由此产生的复杂财产树镜像了我们对现实世界房地产的思考方式。

**半同质化代币 (Semi-fungible tokens)** 模糊了同质化与唯一性之间的界限。活动门票在开始时可能是相同的（同质化），但在使用后会变得唯一，记录下具体的座位、入场时间和活动细节。游戏物品在未使用时可以堆叠，但一旦被玩家装备，就会获得各自的独立历史。

**灵魂绑定代币 (Soulbound Tokens, SBTs)** 则走向相反的方向：它们被刻意设计为不可转让，旨在代表身份、凭证、成就或声誉，这些应该永久地与特定个人绑定。大学学位 NFT 不应该能够出售给另一个人。

### 按使用案例分类的 NFT 类别

**头像类项目 (PFP Projects)** 代表了像 CryptoPunks、Bored Apes 和 Pudgy Penguins 这样在早期热潮中占据主导地位的系列，它们充当了数字身份象征和社交媒体头像。这些系列经历了爆发式增长，但也随着投机狂热的冷却而从巅峰时期显著回落。例如，截至 2026 年初，Bored Apes 的**地板价 (Floor price)**（该系列中可买到的最便宜 NFT 的价格）较其超过 40 万美元的历史高点下跌了 90% 以上。

一直令外界困惑的问题是，为什么这些数字资产的价格能与房屋或精美艺术品等实体奢侈品相媲美，甚至更高。答案部分在于投机，但主要在于人类对数字部落身份表达的根本需求。在我们日益数字化的世界中，这些数字制品的用途远远超出了其视觉吸引力。

正如佩戴劳力士在物理领域传达了成功和社会地位一样，将无聊猿作为 Twitter 头像则标志着拥有一个排他性数字社区的成员身份。在传统身份象征失去意义的数字空间中，这些图像传达了身份、财富和文化认同。

价值通过自我强化的网络效应积累。当 Jay-Z、小威廉姆斯（Serena Williams）和库里（Steph Curry）等知名人物采用这些头像时，文化相关性会放大，从而带来主流认可。稀缺性发挥了基础作用，因为每个系列中存在的数量总是有限的。

**生成艺术 (Generative Art)**：与 PFP 不同，这一类别专注于由自主系统创作的艺术。像 Art Blocks 这样的平台允许艺术家编写算法，在铸造时执行，生成独特的、通常是复杂的且具有美学驱动力的产物。像 Tyler Hobbs 的 *Fidenzas* 或 Snowfro 的 *Chromie Squiggles* 这样的系列因其艺术价值、历史意义和算法新颖性而受到重视，吸引了与社区导向型 PFP 不同的收藏家群体。

**游戏与虚拟世界 NFT**：这些项目代表了基于区块链的游戏中的数字资产，从 *Axie Infinity* 中的生物到 *The Sandbox* 中的土地。虽然“玩以赚取（P2E）”经济和真正的资产所有权的承诺曾是一个强大的叙事，但大多数项目都在努力创造可持续的经济模式或在初始投机之后留住玩家。

**效用与访问类 NFT**：这些充当数字钥匙，授予持有者访问独家社区、活动、软件或服务的权限。它们正越来越多地被探索用于忠诚度计划和订阅模式，充当一种可验证且可交易的会员证明。

**身份与凭证 NFT** 提议使用区块链技术处理学历证书、认证或专业执照等可验证凭证。旨在代表不可转让成就或声誉的灵魂绑定 NFT 无法被转让。

尽管有各种效用主张，但自 2022 年以来，更广泛的 NFT 市场在交易量和地板价方面都出现了剧烈下滑，绝大多数项目难以维持活跃社区或投机交易之外的实际用途。总体交易量仍仅为巅峰时期的一小部分。但像 CryptoPunks 和 Bored Apes 这样的蓝筹系列保留了文化意义和可观的地板价，即使大量的长尾项目归零。这再次印证了 NFT 市场镜像了传统艺术和收藏品市场：少数成名的品牌持有价值，而其他大多数则不然。

### 供应机制

项目如何管理供应对于理解 NFT 价值和市场动态同样至关重要：

*   **固定供应**创造了绝对稀缺性。著名的 10,000 个 CryptoPunks 永远不会增加，这使得每一个都成为有限集合中已知的一份子。
*   **联合曲线 (Bonding curves)** 使用算法定价，每次铸造（Mint）的成本都会递增。一个系列可能从 0.1 ETH 开始，每次铸造增加 0.01 ETH，因此第 50 个代币的成本为 0.59 ETH。这奖励了早期铸造者，同时抑制了后期的投机。
*   **销毁机制 (Burning mechanisms)** 允许代币被永久销毁，从而产生通缩压力。一些系列使用销毁作为演变 NFT 的方式（销毁三个普通物品以铸造一个稀有物品）或获得独家利益的手段。

## 第三节：技术基础

### ERC-721：规则书

以太坊上绝大多数 NFT 都遵循 ERC-721 标准，该标准定义了独特代币在智能合约层级如何运作。虽然 ERC-20（第二章介绍）为每个单位都相同的同质化代币创建了标准，但 ERC-721 为每个都唯一的非同质化代币做了同样的工作。其核心非常简单，只有每个 NFT 合约必须实现的几个基本功能：

*   `ownerOf(tokenId)`：“谁拥有 \#1234 号 NFT？”
*   `transferFrom(from, to, tokenId)`：“将 \#1234 号 NFT 从爱丽丝转移给鲍勃。”
*   `approve(to, tokenId)`：“爱丽丝授权鲍勃转移她的 \#1234 号 NFT。”
*   `setApprovalForAll(operator, approved)`：“爱丽丝授权该市场转移她所有的 NFT。”

该标准还包括用于元数据的可选扩展（链接到那些包含名称、描述和图像的 JSON 文件）和枚举扩展（让应用程序能够发现一个系列中的所有代币）。

### ERC-1155：多代币标准

虽然 ERC-721 处理唯一代币，但 ERC-1155 采用了更灵活的方法。它允许单个智能合约同时管理同质化和非同质化代币，这对于既需要唯一物品（具有独立历史的传说级武器）又需要同质化资源（可互换的金币）的游戏生态系统特别强大。

ERC-1155 引入了批量操作：与其为每个代币转移进行单独交易，不如在一次交易中转移几十个代币，从而大幅降低 Gas 费。这种效率使其成为需要处理大量多样资产的区块链游戏和应用程序的首选标准。

### 安全与常见骗局

这些强大的代币标准实现了复杂的功能，但也创造了安全漏洞。授权功能（特别是 `setApprovalForAll`）授予了广泛的权限，骗局正积极利用这一点。由于区块链交易是不可逆的，用户必须警惕不断的威胁：

*   **网络钓鱼攻击**：骗子创建官方网站的逼真副本，或在 Discord 和 X（原 Twitter）上发送具有欺骗性的链接，诱导用户将钱包连接到恶意网站进行“免费铸造”或“空投”。
*   **钱包清道夫 (Wallet Drainers)**：更高级的骗局涉及诱导用户签署看似合法的交易（如签名请求），但实际上是恶意负载，授权攻击者清空受害者钱包中所有有价值的资产，包括 NFT 和代币。

这些风险强调了自我托管的核心原则：警惕至上。最佳实践——例如使用硬件钱包存储高价值资产，并使用单独的“烧录（Burner）”钱包从新项目中铸造——已成为安全探索该领域的必备技能。

### 发行策略

当项目发布 NFT 时，它们面临着与任何稀缺资源挑战相同的难题：如何在防止机器人和恶意行为者主导销售的同时实现公平分配。

发行模式为此不断演进。**公平发行**以先到先得的方式为每个人提供相同的价格，尽管这些往往被自动化机器人主导。**荷兰拍 (Dutch auctions)** 以高价开始，价格随时间下降，直到需求与供应匹配，这使其对机器人的操纵更具抵抗力。**白名单 (Allowlists)** 授予预先批准的钱包早期访问权限，在公售前奖励社区建设和参与。一些项目也使用前文提到的联合曲线，价格随每次铸造而上涨。

### Solana NFT：一个平行生态系统

Solana（其架构在第三章介绍）发展了很大程度上独立于以太坊的 NFT 生态系统，使用了不同的标准和工具。Metaplex 框架成为大多数 Solana NFT 的基础，处理元数据、系列和版税配置。较新的标准增加了对 NFT 如何转让和使用的更精细控制。

市场格局的演变也大不相同。Magic Eden 凭借用户友好的体验和发行工具主导了早期交易量，而 Tensor 则凭借基于特征的出价和自动化定价池等高级功能吸引了专业交易者。与 OpenSea 长期主导以太坊不同，Solana 的 NFT 流动性一直集中在原生市场。

Solana 独有的一项重大创新是**压缩 NFT (Compressed NFTs)**（第三章介绍了状态压缩），它允许以不到一美分的价格铸造数百万个 NFT。其原理是仅在链上存储加密摘要，而将详细数据保留在链下，从而大幅降低成本。这项技术解锁了大规模空投、忠诚度计划和游戏资产等在其他网络上成本高昂的使用案例。

版税情况镜像了以太坊的演变。市场竞争促使创作者费用趋向于可选，虽然一些 NFT 标准试图在合约层级强制执行版税，但执行仍取决于市场的配合。

结合 Solana 的快速执行和低廉费用，这些特性创造了高频交易文化和频繁的价格变动。著名的系列包括 Solana Monkey Business、Mad Lads 和 Claynosaurz，每一个都反映了 Solana 低成本、实验驱动的文化。

## 第四节：NFT 实际在哪里交易

### 市场大战

NFT 市场最初只是简单的挂单网站，但很快演变为复杂的金融基础设施。OpenSea 凭借先发优势和最简易的用户体验主导了早期市场。但在 NFT 鼎盛时期，OpenSea 变得自满且创新缓慢，为竞争对手创造了机会。

竞争围绕着一个根本性的技术缺陷而加剧。NFT 版税从未被写入核心的 ERC-721 和 ERC-1155 代币标准中，这意味着创作者费用无法自动强制执行。虽然一种名为 **ERC-2981** 的新标准允许合约建议版税金额，但实际支付这些版税仍然是完全可选的。这个技术空白给了市场一个强大的竞争手段：它们可以干脆忽略创作者费用。

大多数 NFT 系列将创作者版税设定在 5-10% 之间，传统上由买家在购买价格之外支付。新兴市场 Blur 发现了一个利用这一弱点的机会。它们以三部分战略启动。首先，它们为专业交易者构建了工具，包括高级投资组合管理、实时价格喂价和精细化筛选。其次，它们将版税设为可选，仅要求向创作者支付最低 0.5% 的费用。第三，它们通过奖励用户 BLUR 代币来激励交易活动。

OpenSea 的反应反复无常且最终归于无效。它们通过运营商过滤器（Operator Filter，于 2022 年 11 月推出）对新系列强制执行全额版税，但无法对旧系列强制收费。交易者自然流向了 Blur 的低费率结构，OpenSea 的市场主导地位开始瓦解。

随着聚合器协议（Aggregator protocols）的兴起，情况变得更加复杂。这些平台，包括 Gem 和 Genie，解决了另一个问题：市场碎片化。它们检查多个市场的价格，并在用户能获得最佳交易的地方自动执行。这项创新在无意中放大了 Blur 的优势。由于聚合器将用户路由到成本最低的市场，Blur 的费用折扣变成了吸引越来越多交易量的结构性优势。这一基础设施层的价值在两个聚合器被迅速收购时变得显而易见：Gem 被 OpenSea 收购，Genie 被 Uniswap 收购。此后，随着交易活动重新向 OpenSea 整合，跨市场路由的需求减少，聚合器的重要性有所下降。

Blur 的策略起初奏效了。到 2023 年 2 月，Blur 在交易量上超过了 OpenSea。到 2023 年 8 月，这种竞争促使 OpenSea 彻底放弃了其版税强制政策。但 Blur 的统治地位被证明是暂时的。其大部分交易量是由代币激励和空投刷量（Airdrop farming）驱动的，而非有机需求，随着这些激励枯竭以及 NFT 交易量全线崩跌，Blur 的结构性优势变得不再重要。到 2025 年，OpenSea 重新夺回了以太坊 NFT 市场的主导地位，这是在 Blur 崛起巅峰时期很少有人能预料到的逆转。

### 定价机制

理解 NFT 如何定价需要掌握几个与传统市场不同的关键概念。

任何 NFT 系列中最受关注的指标是**地板价**。它充当该系列的基准估值，但它可能具有很强的误导性。具有稀有特征的 NFT 售价可能比地板价高出 10 倍甚至更多。例如，一只拥有金色皮毛和激光眼的无聊猿，其价值远高于一只拥有普通棕色皮毛和正常眼睛的猿。

这种差异性创造了对更先进定价方法的需求。**基于特征的定价**作为一种解决方案出现，它考虑到每个 NFT 的个体特征，而不是将系列中的所有作品视为等同。

另一项创新——**针对整个系列的出价 (Collection-wide bidding)**，解决了另一个问题：流动性不足。买家无需针对某个特定的 NFT 出价，而是可以针对任何符合特定标准的 NFT（如“任何带激光眼的无聊猿”）出价。这改善了卖家的流动性，使价格发现更有效率。其代价是哲学上的：它使所谓的独特资产变得商品化了。Blur 曾尝试通过奖励忠诚度积分来推广特征层级的出价，OpenSea 也增加了对系列和特征报价的支持，但在实践中，特征出价从未获得广泛采用。大多数交易活动仍然集中在扫货（Floor sweeps）和针对整个系列的出价，而非细颗粒度的基于特征的策略。

这种独特与同质化之间的张力处于 NFT 的核心。从市场功能到定价机制，每一个设计决策都反映了这一根本悖论，并塑造了关于 NFT 如何被使用和估值的一切。
