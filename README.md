关于本书

《加密货币的运作原理》是一本全面的技术书籍，解释了加密货币的实际运作方式，从比特币的UTXO模型到抗量子密码学。全书共15章，包含一篇前言，超过9万字，涵盖了从基础概念到加密生态系统最新发展的所有内容。

本仓库包含本书的完整文本，本书开源，可免费阅读、下载和分享。此版本发布在GitHub上，旨在收集社区的反馈。最佳的编辑和修正意见将被纳入正式的第一版，该版本将在未来几个月内以免费电子书的形式在亚马逊和其他平台上发布。

本书的缘起

当我尝试推荐一本能够为读者打下坚实加密货币基础的单一资源时，我却发现无从下手。有些书籍深入探讨某一协议，却忽略了其他协议。YouTube 和 X 平台充斥着伪装巧妙的广告。最优秀的研究成果分散在数十个付费网站上。学术论文严谨，但过于抽象，难以实际应用。

所有这些都存在不足：要么过于狭隘，要么过于片面，要么过于深奥，要么过于分散，或者仅仅是因为缺乏指导而难以入门。

本书正是我尝试解决这些问题的尝试。

本书特色

全面覆盖：共15章，涵盖比特币、以太坊、Solana、DeFi、MEV、稳定币、托管、市场结构、治理、NFT、DePIN、量子抗性等内容。

与时俱进：加密货币发展日新月异，大多数书籍在出版前就已经过时。本书持续更新，包含最新的数据、协议和市场动态。

结构化学习：每章内容都以前一章为基础，构建出一个连贯的故事，难度逐渐递增。

专家审阅：每章都由在该领域拥有丰富经验的专家审阅。

客观评价：技术概念解释清晰易懂，不故弄玄虚，并探讨了其优势和局限性。

开源：全书免费阅读和分享。基础教育不应被付费墙阻隔。

本书适合哪些读者？

本书面向所有希望了解技术运作原理的人士，而不仅仅是交易员或投资者。本书假定读者具有求知欲，但并不要求具备任何专业知识。

如果你认真阅读并吸收其中的关键思想，你最终会比现在大多数全职从事加密货币行业的人知道得更多。
